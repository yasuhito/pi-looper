# 機能: Git 管理ファイルが作業用一時ディレクトリにある場合、自動チェックを実行しない

`.deadloop` と `.pi-subagents` は、deadloop や関連エージェントが一時ファイルを置くための
作業用一時ディレクトリであり、Git 管理ファイルを置かないことを想定している。

deadloop は、自動チェックの間だけこれらのディレクトリをリポジトリの外へ移し、
自動チェックの終了後に元へ戻す。

これらのディレクトリに Git 管理ファイルがある場合、deadloop は想定外の状態と判断し、
ディレクトリを移動せず、自動チェックも実行しない。

## シナリオ: `.deadloop` に Git 管理ファイルがある場合は自動チェックを実行しない

* 前提 deadloop が自動チェックするプロジェクトがある
* かつ `.deadloop` ディレクトリに Git 管理ファイルがある
* もし deadloop が自動チェックを開始しようとする
* ならば deadloop は自動チェックを実行しない
